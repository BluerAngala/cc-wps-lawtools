## 1️⃣ 模板管理

### 使用场景
需要快速起草法律文书时（起诉状、答辩状、合同等）

### 操作步骤
```
1. 点击 WPS 功能区的"模板管理"按钮
2. 选择需要的模板分类（诉讼文书/合同模板/律师函/自定义）
3. 浏览模板列表，点击"应用到当前文档"或"新建文档"
4. 根据需要修改模板中的变量内容（如 {{原告姓名}}）
```

### 保存自定义模板
```
1. 编辑好文档内容
2. 点击"保存当前文档为模板"按钮
3. 填写模板名称、分类、描述
4. 点击"保存"
```

### 内置模板列表
- ✅ 民事起诉状
- ✅ 民事答辩状
- ✅ 代理词
- ✅ 劳动合同
- ✅ 律师函

## 2️⃣ 合同风险扫描

### 使用场景
需要快速了解合同潜在风险，但不想修改原文档

### 操作步骤
```
1. 在 WPS 中打开需要扫描的合同文档
2. 点击 WPS 功能区的"风险扫描"按钮
3. 选择扫描策略：
   - 全文扫描（推荐）：适合中短篇合同
   - 分段扫描：适合长篇合同
4. 点击"开始扫描"按钮
5. 等待 AI 分析完成（会显示进度）
6. 查看扫描结果：
   - 合同类型识别
   - 风险统计
   - 审查清单
   - 检测到的问题
   - 风险提示
7. 可点击"导出报告"复制到剪贴板
```

### 扫描结果说明
- **风险等级**：
  - 🔴 高风险：需要立即处理的严重问题
  - 🟡 中风险：建议关注的潜在问题
  - 🔵 低风险：可选择性处理的轻微问题

- **审查清单**：
  - ✓ 已检测到相关内容
  - ○ 未检测到相关内容

## 3️⃣ 信息脱敏

### 使用场景
需要分享文档但要保护敏感信息（身份证号、手机号等）

### 操作步骤
```
1. 在 WPS 中打开需要脱敏的文档
2. 点击 WPS 功能区的"信息脱敏"按钮
3. 点击"扫描文档"按钮
4. 系统自动检测敏感信息并列出：
   - 身份证号：显示前6位和后4位，中间用*代替
   - 手机号：显示前3位和后4位，中间用*代替
   - 邮箱：只显示首字母和域名
   - 银行卡号：显示前4位和后4位
   - 姓名：保留首尾，中间用*代替
5. 默认全选，可取消不需要脱敏的项
6. 点击"一键脱敏"应用到文档
```

### ⚠️ 重要提示
- 脱敏操作会直接修改文档内容
- 脱敏后无法自动恢复
- **建议先备份原文档再进行脱敏**

### 脱敏示例
```
原始内容：
姓名：张三
身份证号：110101199001011234
手机号：13800138000

脱敏后：
姓名：张*三
身份证号：110101********1234
手机号：138****8000
```

## 💡 使用技巧

### 模板管理
1. 可以将常用的文书格式保存为模板，提高工作效率
2. 模板变量使用 `{{变量名}}` 格式，便于快速替换
3. 建议按照业务类型分类管理模板

### 风险扫描
1. 首次扫描建议使用"全文扫描"，更全面
2. 长文档（>5000字）建议使用"分段扫描"，更稳定
3. 扫描结果可导出为文本，便于存档
4. 与"合同审查"功能配合使用效果更佳：
   - 先用"风险扫描"快速了解整体情况
   - 再用"合同审查"进行深度分析和批注

### 信息脱敏
1. 脱敏前务必备份原文档
2. 可以先扫描查看检测结果，确认无误后再应用
3. 如有不需要脱敏的信息，取消勾选即可
4. 建议在文档定稿后再进行脱敏

## ❓ 常见问题

### Q1: 模板保存后在哪里？
A: 自定义模板保存在浏览器的本地存储（localStorage）中，只在当前电脑有效。

### Q2: 风险扫描需要多长时间？
A: 通常 1-3 分钟，取决于文档长度和网络状况。

### Q3: 脱敏后能恢复吗？
A: 不能自动恢复，请务必提前备份原文档。

### Q4: 为什么点击按钮没反应？
A: 请确保：
- 在 WPS 环境中运行
- 已打开文档（某些功能需要）
- 网络连接正常（AI 功能需要）

### Q5: 风险扫描和合同审查有什么区别？
A: 
- **风险扫描**：只分析不修改，快速了解风险
- **合同审查**：深度分析+批注/修订，可修改文档
