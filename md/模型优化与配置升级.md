# 模型优化与配置升级

## 🎯 解决的问题

### 原始问题
```
timeout of 120000ms exceeded
POST https://api.siliconflow.cn/v1/chat/completions net::ERR_TIMED_OUT
```

**问题分析**：
1. ❌ 使用的模型（inclusionAI/Ling-1T）响应速度慢
2. ❌ 超时时间设置为 120 秒不够
3. ❌ 无法方便地切换模型

## ✅ 优化方案

### 1. 更换推荐模型

#### 原模型
- `inclusionAI/Ling-1T`（速度慢，容易超时）
- `moonshotai/Kimi-K2-Instruct-0905`（长文本但速度一般）

#### 新推荐模型（按速度排序）
```
🏆 推荐（速度快）
├── Qwen/Qwen2.5-7B-Instruct ⭐⭐⭐⭐⭐ 最推荐
├── Qwen/Qwen2.5-14B-Instruct ⭐⭐⭐⭐
└── Pro/THUDM/glm-4-9b-chat ⭐⭐⭐⭐

💪 高级（功能强）
├── Qwen/Qwen2.5-32B-Instruct
├── Qwen/Qwen2.5-72B-Instruct
├── deepseek-ai/DeepSeek-V3
└── deepseek-ai/DeepSeek-R1
```

### 2. 增加超时时间
- **原配置**：120 秒（2 分钟）
- **新配置**：180 秒（3 分钟）
- **原因**：给长文档审查留出足够时间

### 3. 模型列表动态获取

#### 新增功能
- ✅ 自动从 SiliconFlow API 获取最新模型列表
- ✅ 下拉选择器，支持搜索过滤
- ✅ 按推荐顺序排序
- ✅ 显示模型标签（推荐/高级）
- ✅ 一键刷新模型列表

#### UI 改进
```
模型选择器
├── 下拉列表（可搜索）
├── 推荐标签显示
├── 刷新按钮
├── 模型数量提示

```

## 📝 使用指南

### 1. 首次配置
1. 打开设置页面
2. 输入 API Key
3. 点击"刷新模型列表"（自动获取）
4. 选择推荐模型：`Qwen/Qwen2.5-7B-Instruct`
5. 保存（自动）

### 2. 更换模型
1. 打开设置页面
2. 在模型下拉框中搜索或选择
3. 推荐选择：
   - **快速审查**：Qwen2.5-7B
   - **平衡性能**：Qwen2.5-14B
   - **复杂合同**：Qwen2.5-72B 或 DeepSeek-V3

### 3. 调整超时时间
- **短文档**（<5页）：60-120 秒
- **中等文档**（5-20页）：120-180 秒
- **长文档**（>20页）：180-300 秒
