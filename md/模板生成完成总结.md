## 🚀 使用方式

### 重新生成模板
```bash
pnpm create-templates
```

### 添加新模板
1. 编辑 `scripts/create-template-docs.js`
2. 在 `templates` 对象中添加新模板定义
3. 更新 `public/templates/templates.json` 配置
4. 运行 `pnpm create-templates`

## 🎯 技术优势

### 为什么选择 Node.js + docx？

1. **跨平台**
   - 无需安装 Microsoft Word
   - Windows、Mac、Linux 都可运行

2. **自动化**
   - 一键生成所有模板
   - 格式统一标准

3. **可维护**
   - 代码即文档
   - 易于修改和扩展

4. **版本控制**
   - 模板定义在代码中
   - 便于追踪变更

5. **CI/CD 友好**
   - 可集成到构建流程
   - 自动化部署

## 🔧 脚本工作原理

```javascript
// 1. 定义模板结构
const templates = {
  '民事起诉状.docx': {
    title: '民事起诉状',
    sections: [
      { type: 'paragraph', content: '...' },
      { type: 'heading', content: '...' }
    ]
  }
}

// 2. 使用 docx 库创建文档
const doc = new Document({
  sections: [{ children: paragraphs }]
})

// 3. 生成并保存文件
const buffer = await Packer.toBuffer(doc)
fs.writeFileSync(filePath, buffer)
```
