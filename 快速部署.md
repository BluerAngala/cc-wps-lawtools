## 打包


```
wpsjs publish
```


提示输入服务器地址： （注意，一定要  / 结尾）

```
https://env-00jxg9mus2ok-static.normal.cloudstatic.cn/wps-addon-build/
```


其实就是将 wps-addon-build 上传到服务器，提供可以访问到的地址就行

然后选择离线模式 ， 多用户

![PixPin_2025-10-08_18-52-01](https://chnotes.oss-cn-guangzhou.aliyuncs.com/siyuan/PixPin_2025-10-08_18-52-01-20251008185202-pe2qxql.png)

执行完毕后，会生成两个文件夹：

wps-addon-build 是具体程序执行代码，全部要上传

wps-addon-publish  下只有一个 publish.html ，用户分发给用户访问，提供下载地址。


服务器用unicloud-前端部署就行

![PixPin_2025-10-08_18-56-35](https://chnotes.oss-cn-guangzhou.aliyuncs.com/siyuan/PixPin_2025-10-08_18-56-35-20251008185655-2df5333.png)
