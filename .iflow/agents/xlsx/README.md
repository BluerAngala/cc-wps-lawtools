XLSX Workflow - Excel电子表格智能处理工作流

一键安装：
在iFlow CLI执行命令（"iflow workflow..."替换为安装指令）：
示例：!iflow workflow add "xlsx-aBcDef"
安装后重启iFlow CLI，输入 `/xlsx` 使用。

核心功能：
1. 创建电子表格
创建Excel文件，支持公式格式。
示例：/xlsx 创建销售报表含月度汇总
示例：/xlsx 制作财务模型含利润表

2. 数据分析
读取分析Excel数据。
示例：/xlsx 分析sales.xlsx趋势
示例：/xlsx 统计客户数据

3. 公式管理
创建动态Excel公式。
示例：/xlsx 添加SUM计算总额
示例：/xlsx 创建增长率公式

4. 编辑表格
修改Excel保留公式。
示例：/xlsx 修改budget.xlsx
示例：/xlsx 更新report.xlsx

5. 财务建模
创建专业财务模型。
示例：/xlsx 建立财务预测
示例：/xlsx 创建估值模型

6. 数据可视化
在Excel创建图表。
示例：/xlsx 添加柱状图
示例：/xlsx 创建折线图
