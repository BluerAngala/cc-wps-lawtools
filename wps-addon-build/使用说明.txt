WPS LawTools v1.2.0 安装指南
========================================

【Windows 安装】

1. 双击 WPS_LawTools_Windows_v1.2.0_setup.exe
2. 按照安装向导点击"下一步"完成安装
3. 打开 WPS Office，功能区将出现新选项卡
4. 如未出现，请重启 WPS Office

卸载方式：
  控制面板 → 程序与功能 → 找到本程序 → 卸载
  或在安装目录运行 uninstall.exe

【macOS 安装】

1. 双击 WPS_LawTools_Mac_v1.2.0.dmg 打开
2. 在弹出的窗口中双击 "安装LawTools.command"
3. 按照终端提示完成安装
4. 完全退出 WPS (Cmd+Q) 后重新打开

常见问题：
  - 提示"无法打开"：右键 → 打开；或在终端 chmod +x 后再双击
  - 安装后看不到插件：请完全退出 WPS (Cmd+Q) 后重新打开

【AI 服务配置（必填）】

安装完成后，打开 WPS 中的插件，进入"设置"页面配置 AI 服务：

本工具默认使用硅基流动（SiliconFlow）作为 AI 服务提供商，
按以下步骤获取 API Key：

1. 注册账号
   打开 https://cloud.siliconflow.cn/i/WFoChvZf 注册硅基流动账号

2. 实名认证
   登录后完成实名认证（需上传身份证，一般几分钟即可通过）

3. 创建 API Key
   进入"API 密钥"页面 → 点击"新建密钥" → 复制生成的 Key

4. 填写 Key
   回到插件设置页面，将复制的 Key 粘贴到"API Key"输入框
   API 地址保持默认（https://api.siliconflow.cn/v1），无需修改

填写完成后点击"刷新"按钮测试模型列表是否正常加载。
