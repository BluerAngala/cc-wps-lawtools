# 合同预审功能使用说明

## 功能概述

合同预审功能是一个基于AI的智能合同审查工具，能够自动识别合同中的法律风险点，并提供专业的修改建议。

## 主要特性

### 1. 智能风险识别
- 自动识别违约责任条款问题
- 检查付款条件的合理性
- 分析争议解决机制的完整性
- 评估权利义务的平衡性

### 2. 双重执行模式

#### 批注模式
- 在文档中添加详细的批注说明
- 标注风险点和改进建议
- 保持原文档结构不变

#### 修订模式
- 直接修改有问题的条款内容
- 启用WPS修订模式跟踪更改
- 提供修改原因说明

## 使用步骤

### 1. 打开合同文档
在WPS中打开需要审查的合同文档

### 2. 启动合同预审
1. 在WPS插件面板中找到"合同审查"功能
2. 展开"AI合同预审"规则
3. 配置预审参数：
   - **规则名称**：设置关注的重点条款（如违约责任、付款条件等）
   - **审查要求**：指定具体的审查标准（如识别法律风险、条款不明确等）
   - **执行动作**：选择"批注"或"修订"模式

### 3. 执行预审
点击"执行"按钮，AI将自动分析合同内容并生成审查结果

### 4. 查看结果
- **批注模式**：在文档中查看添加的批注
- **修订模式**：查看修订痕迹和修改建议

## 配置说明

### 规则名称示例
```
重点关注违约责任、付款条件、争议解决等关键条款
```

### 审查要求示例
```
识别法律风险、条款不明确、权利义务不对等等问题
```

## 技术实现

### 核心组件
1. **promptGenerator.js** - 动态生成AI提示词
2. **siliconflow.js** - 处理AI接口调用
3. **taskpane.js** - WPS文档操作逻辑
4. **ContractReview.vue** - 用户界面组件

### 数据流程
1. 用户配置预审参数
2. 提取WPS文档内容
3. 生成专业的AI提示词
4. 调用AI接口进行分析
5. 解析AI返回结果
6. 根据执行模式添加批注或修订

## 注意事项

1. **文档格式**：支持WPS文字文档格式
2. **网络连接**：需要稳定的网络连接以访问AI服务
3. **文档权限**：确保对文档有编辑权限
4. **AI限制**：AI分析结果仅供参考，重要合同建议人工复核

## 常见问题

### Q: 预审结果不准确怎么办？
A: 可以调整规则名称和审查要求，使其更具体和针对性。

### Q: 批注模式没有添加批注？
A: 检查AI是否识别到相关关键词，可能需要调整规则名称。

### Q: 修订模式没有修改内容？
A: 确认文档没有被保护，且AI识别到需要修订的内容。
