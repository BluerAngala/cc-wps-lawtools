# WPS 对话框使用指南

## 简介

`dialogHelper.js` 提供了一个通用的 WPS 对话框创建方法，用于在 WPS 环境中弹出自定义页面。

## 核心方法

### `showWPSDialog(routePath, options)`

显示 WPS 原生对话框的通用方法。

#### 参数

- **routePath** `{string}` - 必需，路由路径
  - 示例: `'/'`, `'/settings'`, `'/contractreview'`
  
- **options** `{Object}` - 可选，配置项
  - `width` `{number}` - 对话框宽度（逻辑像素），默认 `600`
  - `height` `{number}` - 对话框高度（逻辑像素），默认 `450`
  - `modal` `{boolean}` - 是否模态，默认 `true`
  - `hasCaption` `{boolean}` - 是否有标题栏，默认 `false`（无边框）
  - `resizeEdge` `{number}` - 缩放边框宽度，默认 `2`
  - `caption` `{string}` - 标题文字，默认 `''`

#### 使用示例

```javascript
import { showWPSDialog } from '@/utils/dialogHelper.js'

// 基础用法 - 使用默认配置
showWPSDialog('/settings')

// 自定义尺寸
showWPSDialog('/settings', {
  width: 800,
  height: 600
})

// 完整配置
showWPSDialog('/custom-page', {
  width: 700,
  height: 500,
  modal: true,
  hasCaption: false,
  resizeEdge: 2,
  caption: ''
})
```

## 快捷方法

为了向后兼容和方便使用，提供了以下快捷方法：

### `showWelcomeDialog()`

显示欢迎页面对话框（路由: `/`，尺寸: 600x450）

```javascript
import { showWelcomeDialog } from '@/utils/dialogHelper.js'

showWelcomeDialog()
```

### `showSettingsDialog()`

显示设置页面对话框（路由: `/settings`，尺寸: 600x500）

```javascript
import { showSettingsDialog } from '@/utils/dialogHelper.js'

showSettingsDialog()
```

## 添加新的对话框

### 步骤 1: 创建页面组件

在 `src/views/` 目录下创建新页面，参考 `HomePage.vue` 或 `SettingsPage.vue` 的结构：

```vue
<template>
  <n-config-provider>
    <n-message-provider>
      <div class="min-h-screen bg-gradient-to-br from-blue-500 to-purple-600 flex items-center justify-center p-0">
        <n-card class="w-full h-full backdrop-blur-md rounded-none shadow-none m-0" :bordered="false">
          <!-- 关闭按钮 -->
          <template #header>
            <div class="relative">
              <n-button quaternary circle size="small" class="absolute -right-2 -top-2 z-10" @click="closeDialog">
                <template #icon>
                  <n-icon size="16">
                    <svg viewBox="0 0 24 24">
                      <path fill="currentColor" d="M18 6L6 18M6 6L18 18" stroke="currentColor" stroke-width="2" stroke-linecap="round" />
                    </svg>
                  </n-icon>
                </template>
              </n-button>
            </div>
          </template>

          <!-- 页面内容 -->
          <n-space vertical size="large" class="px-6 py-4">
            <div class="text-center">
              <h1 class="text-3xl font-bold mb-4">标题</h1>
              <n-text class="text-lg">内容</n-text>
            </div>
          </n-space>
        </n-card>
      </div>
    </n-message-provider>
  </n-config-provider>
</template>

<script setup>
const closeDialog = () => {
  window.close()
}
</script>
```

### 步骤 2: 添加路由

在 `src/router/index.js` 中添加路由配置：

```javascript
{
  path: '/your-page',
  name: '页面名称',
  component: () => import('../views/YourPage.vue')
}
```

### 步骤 3: 调用对话框

使用通用方法打开：

```javascript
import { showWPSDialog } from '@/utils/dialogHelper.js'

showWPSDialog('/your-page', { width: 600, height: 500 })
```

或在 `dialogHelper.js` 中添加快捷方法：

```javascript
export function showYourPageDialog() {
  showWPSDialog('/your-page', { width: 600, height: 500 })
}
```

## 设计建议

1. **使用全屏渐变背景**：创造沉浸式体验
2. **添加关闭按钮**：位于右上角，方便用户关闭
3. **保持一致的设计风格**：使用 NativeUI 组件和 UnoCSS
4. **响应式设计**：确保在不同尺寸下都能正常显示
5. **无边框设计**：`hasCaption: false` 提供现代化外观

## 注意事项

- 对话框只能在 WPS 环境中使用
- 尺寸单位为逻辑像素，会自动根据设备像素比转换
- 路由路径必须在 `router/index.js` 中已配置
- 关闭对话框使用 `window.close()`
