# 方案管理使用说明

## 功能概述

方案管理功能允许用户为不同类型的合同创建专属的审查方案，快速切换使用，提高工作效率。

## 功能特点

- ✅ 多方案管理：为不同合同类型创建独立方案
- ✅ 快速切换：一键切换不同审查方案
- ✅ 自动保存：规则修改自动保存到当前方案
- ✅ 方案复制：基于现有方案快速创建新方案
- ✅ 规则统计：显示每个方案包含的规则数量
- ✅ 向后兼容：自动迁移旧配置到方案系统

## 使用场景

### 场景1：采购合同审查
创建"采购合同方案"，配置采购合同特有的关键词和审查规则。

### 场景2：劳动合同审查
创建"劳动合同方案"，配置劳动合同相关的审查要点。

### 场景3：服务合同审查
创建"服务合同方案"，针对服务类合同的特殊条款进行配置。

## 操作指南

### 1. 查看当前方案
在"智能文档处理"页面，选择"关键词模式"或"AI+律师模式"，即可看到方案选择器。

### 2. 切换方案
点击方案下拉选择器，选择要使用的方案。系统会自动加载该方案的规则列表。

### 3. 新建方案
1. 点击"新建"按钮
2. 输入方案名称（如：采购合同方案）
3. 选择初始化方式：
   - 创建空方案：从零开始配置
   - 基于当前方案复制：复制当前方案的所有规则
4. 点击"确定"

### 4. 编辑方案
1. 选择要编辑的方案
2. 点击"编辑"按钮
3. 修改方案名称
4. 点击"确定"

### 5. 复制方案
1. 选择要复制的方案
2. 点击"复制"按钮
3. 输入新方案名称
4. 点击"确定"

### 6. 删除方案
1. 选择要删除的方案
2. 点击"删除"按钮
3. 确认删除操作

**注意：** 至少需要保留一个方案，最后一个方案无法删除。

### 7. 修改规则
在配置表单中添加、修改或删除规则，系统会自动保存到当前激活的方案。

## 数据存储

方案数据存储在 WPS 插件配置文件中（`config.json`），位置：
- Windows: `C:\Users\用户名\AppData\Roaming\Kingsoft\wps\jsaddons\wps_lawtools\config.json`

## 兼容性说明

首次使用方案管理功能时，系统会自动将旧的配置迁移到"默认方案"中，不会丢失任何数据。

## 技术实现

- **组件**: `SchemeSelector.vue` - 方案选择器组件
- **配置管理**: `appConfig.js` - 扩展了方案管理方法
- **数据结构**:
  ```javascript
  {
    keywordSchemes: {
      schemes: [
        {
          id: 'scheme_1',
          name: '采购合同方案',
          type: 'keyword',
          rules: [...],
          createdAt: '2024-01-01',
          updatedAt: '2024-01-01'
        }
      ],
      activeSchemeId: 'scheme_1'
    }
  }
  ```

## 常见问题

### Q: 切换方案后，之前的规则会丢失吗？
A: 不会。每个方案的规则都是独立保存的，切换方案只是加载不同的规则集。

### Q: 可以创建多少个方案？
A: 理论上没有限制，但建议根据实际需要创建，避免方案过多难以管理。

### Q: 删除方案后可以恢复吗？
A: 删除操作不可恢复，请谨慎操作。建议在删除前先复制一份备份。

### Q: 方案数据会同步到其他电脑吗？
A: 方案数据存储在本地配置文件中，不会自动同步。如需在多台电脑使用，可以手动复制配置文件。
