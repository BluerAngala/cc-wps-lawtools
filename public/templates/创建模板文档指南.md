# 创建模板文档指南

## 方法一：使用 PowerShell 脚本自动创建（推荐）

如果你的电脑安装了 Microsoft Word，可以使用自动化脚本：

1. 打开 PowerShell（以管理员身份运行）
2. 进入项目根目录
3. 运行脚本：
```powershell
.\scripts\create-template-docs.ps1
```

脚本会自动创建所有 5 个模板文档，并保存到 `public/templates/` 目录。

## 方法二：手动创建

### 步骤

1. **打开文本文件**
   - 打开 `public/templates/模板内容-请复制到Word.txt`
   - 查看每个模板的内容

2. **创建 Word 文档**
   - 打开 Microsoft Word
   - 新建空白文档

3. **复制内容**
   - 从文本文件复制对应模板的内容
   - 粘贴到 Word 文档中

4. **设置格式**（建议）
   - **标题**：黑体、二号、居中、加粗
   - **一级标题**：黑体、小三、加粗
   - **二级标题**：黑体、四号、加粗
   - **正文**：宋体、小四、首行缩进2字符
   - **行距**：1.5倍
   - **变量占位符**：可设置为红色或蓝色以便识别

5. **保存文档**
   - 文件 → 另存为
   - 保存位置：`public/templates/`
   - 文件名：按配置文件中的名称保存
     - 民事起诉状.docx
     - 民事答辩状.docx
     - 代理词.docx
     - 劳动合同.docx
     - 律师函.docx
   - 文件类型：Word 文档 (*.docx)

## 格式建议

### 页面设置
- 纸张：A4
- 页边距：上下 2.54cm，左右 3.17cm

### 字体设置
- 标题：黑体、二号（22磅）、居中、加粗
- 一级标题：黑体、小三（15磅）、加粗
- 二级标题：黑体、四号（14磅）、加粗
- 正文：宋体、小四（12磅）
- 变量占位符：可用红色或蓝色标注

### 段落设置
- 首行缩进：2字符
- 行距：1.5倍
- 段前段后：0.5行

## 变量占位符说明

模板中使用 `{{变量名}}` 格式的占位符，用户应用模板后需要手动替换。

常用变量：
- `{{原告姓名}}`、`{{被告姓名}}`
- `{{性别}}`、`{{出生日期}}`、`{{民族}}`
- `{{住址}}`、`{{联系电话}}`
- `{{身份证号}}`
- `{{法院名称}}`、`{{案由}}`
- `{{日期}}`

## 验证

创建完成后，检查：
1. 文件名是否正确
2. 文件格式是否为 .docx
3. 文件是否保存在 `public/templates/` 目录
4. 内容是否包含所有变量占位符
5. 格式是否美观规范

完成后刷新应用，即可在模板管理中看到这些模板！
